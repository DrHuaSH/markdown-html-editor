# 🚀 快速在线使用指南

## 🌟 最简单方式：独立版本

### 📥 下载使用
1. **下载文件**: 右键点击 [standalone-editor.html](./standalone-editor.html) → "另存为"
2. **双击打开**: 在浏览器中双击文件
3. **立即使用**: 无需安装，完全离线工作

### ✨ 优势
- ✅ **零安装**: 无需安装任何软件
- ✅ **离线使用**: 不需要网络连接
- ✅ **完整功能**: 包含所有编辑器功能
- ✅ **跨平台**: Windows、macOS、Linux 通用

## 🌐 在线部署版本

### GitHub Pages (推荐)
```bash
# 1. Fork 项目到你的 GitHub 账号
# 2. 在仓库设置中启用 Pages
# 3. 访问: https://your-username.github.io/markdown-html-editor/
```

### Netlify (最快部署)
1. 访问 [netlify.com](https://netlify.com)
2. 拖拽 `build` 文件夹到页面
3. 获得即时访问链接

### Vercel (专业部署)
```bash
npm i -g vercel
vercel
# 按提示完成配置
```

## 📱 功能特色

### HTML 可视化编辑
- **直接编辑**: 点击预览区域任何文字直接编辑
- **格式化工具**: 粗体、斜体、链接、标题等
- **撤销重做**: Ctrl+Z / Ctrl+Y 快捷键
- **拖拽滚动**: Shift+拖拽自由滚动

### Markdown 编辑
- **实时预览**: 左侧编辑，右侧预览
- **语法高亮**: 专业代码编辑体验
- **快捷操作**: 完整的编辑功能

### 文件操作
- **导入**: 支持 .md、.html、.txt 文件
- **导出**: 保存为对应格式
- **转换**: Markdown ↔ HTML 互转

## 💡 使用技巧

1. **添加书签**: 将在线版本添加到浏览器书签
2. **移动端**: 添加到主屏幕获得原生应用体验
3. **快捷键**: 熟练使用 Ctrl+Z/Y 提高效率
4. **定期保存**: 使用保存按钮备份重要内容
